## 如何使用
首先需要查看您的操作系统版本和发行版类型

1.打开终端，在终端内输入
`lsb_release -a`
查看系统版本

若`Distributor ID:`后方显示`Kali`则点击<a href="https://github.com/1216692740/Fast-Install-Chinese-Debian-Apps/blob/main/help/kali.md" title="Kali套件安装">此处</a>查看教程
